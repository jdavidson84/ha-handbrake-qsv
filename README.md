# HandBrake QSV Encoder (Home Assistant Add-on)

This repository provides a custom Home Assistant add-on that runs
a headless HandBrake server using Intel Quick Sync Video (QSV)
for hardware-accelerated encoding, including HEVC and AV1 on supported CPUs.

## Installation

1. Go to Home Assistant → Settings → Add-ons → Add-on Store.
2. Click the top-right menu (⋮) → Repositories.
3. Add this URL:
   ```
   https://github.com/jdavidson84/ha-handbrake-qsv
   ```
4. Install the **HandBrake QSV Encoder** add-on.
5. Configure input/output paths and presets.
